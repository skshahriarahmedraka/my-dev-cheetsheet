service selector label have to match with the replicaset selector labels 



annotation is used to add some meta data  like build version, email,phone number 

`replicaset-defination.yml`

```
apiVersion: apps/v1 
kind: ReplicaSet
metadata:
    name: simple-webapp
    labels:
        app: App1
        function: front-end
    annotations:
        buildversion: 1.34
spec:
    replica: 3
    selector: 
        matchLabels:
            app: App1
    template:
        metadata:
            labels:    
                app: App1
                function: front-end
        spec:
            containers:
            - name: simple-webapp
              image: simpel/webapp
```
